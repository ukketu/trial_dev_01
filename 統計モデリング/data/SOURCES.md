# Local open data sources

The notebooks read these local CSV files directly so they can run without network access.
Original source repository: https://github.com/mwaskom/seaborn-data

| Dataset | Local file | Original upstream URL |
|---|---|---|
| Palmer Penguins | `data/raw/penguins.csv` | https://raw.githubusercontent.com/mwaskom/seaborn-data/master/penguins.csv |
| Old Faithful geyser | `data/raw/geyser.csv` | https://raw.githubusercontent.com/mwaskom/seaborn-data/master/geyser.csv |
| Tips | `data/raw/tips.csv` | https://raw.githubusercontent.com/mwaskom/seaborn-data/master/tips.csv |
| Auto MPG | `data/raw/mpg.csv` | https://raw.githubusercontent.com/mwaskom/seaborn-data/master/mpg.csv |

`diamonds.csv` was kept as an unused candidate only in the original explanation and is not required by the generated GIF notebooks.
